# MediaPipe BlazeFace (Face Detection)

> **View on MLange Dashboard**: [google/MediaPipe-Face-Detection](https://mlange.zetic.ai/p/google/MediaPipe-Face-Detection?from=use-cases) - Contains generated source code & benchmark reports.

This application showcases a lightweight face detection model using ZETIC.MLange.

## Model Details
- **Model**: Google MediaPipe BlazeFace (short-range)
- **Description**: A lightweight and highly optimized face detection architecture specifically designed for mobile and edge devices.
- **Task**: Face Detection
- **Target Platforms**: Android, iOS
