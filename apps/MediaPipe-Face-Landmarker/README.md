# MediaPipe Face Landmarker

This application showcases the **MediaPipe Face Landmarker** model using ZETIC.MLange.

## Model Details
- **Model**: Google MediaPipe Face Landmarker
- **Description**: This model is based on Google's MediaPipe Face Landmarker, providing high-fidelity face mesh tracking.
- **Task**: Face Landmark Detection
- **Target Platforms**: Android, iOS
