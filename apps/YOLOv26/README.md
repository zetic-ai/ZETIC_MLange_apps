# Team ZETIC YOLOv26 (Nano)

This application showcases the **YOLOv26 Nano** model using ZETIC.MLange.
YOLOv26 is a next-generation object detection model that eliminates the need for Non-Maximum Suppression (NMS), making it highly efficient for edge devices.

## Model Details
- **Model**: YOLOv26 Nano
- **Task**: Object Detection
- **Version**: 3
- **Features**: NMS-Free, Real-time Performance
